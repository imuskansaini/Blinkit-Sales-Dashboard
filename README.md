# Blinkit Sales Dashboard (Power BI)

## Project Overview

This project presents an interactive Blinkit Sales Dashboard developed using Power BI to analyze product sales, pricing, discounts, customer ratings, and delivery performance. The dashboard provides insights into sales trends, product performance, brand performance, and customer satisfaction, enabling data-driven business decisions.

## Dashboard Features

- Total Revenue
- Products Sold
- Total Products
- Average Rating
- Average Discount
- Average Profit Margin
- Sales by Category
- Top 10 Brands by Revenue
- Revenue by City
- Delivery Status Distribution
- Average Rating by Category
- Average Discount by Category

## Tools Used

- Microsoft Power BI
- Power Query
- DAX
- Microsoft Excel
- CSV Dataset

## Key Insights

- Identified the highest revenue-generating product categories.
- Compared sales performance across different brands.
- Analyzed revenue distribution across multiple cities.
- Evaluated delivery performance using delivery status analysis.
- Measured customer satisfaction through average product ratings.
- Examined discount trends and their impact across categories.
